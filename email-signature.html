How to Add to Gmail:

  1. Open this file in your browser
  2. Select and copy the signature table above (between the comment lines)
  3. Go to Gmail Settings → See all settings → General
  4. Scroll down to "Signature"
  5. Click "Create new" signature
  6. Paste using Ctrl+V (NOT right-click paste)
  7. Save changes

Important: Gmail strips complex CSS, so this version uses only basic formatting that Gmail supports.
