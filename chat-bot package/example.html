Wood Tide Travel Chat Widget

This is an example page showing how to integrate the chat widget. Look for the chat button in the bottom-right corner!

Quick Start Guide

  1. Make sure you have EmailJS configured (see README.md)
  2. Update the EmailJS credentials in the script below
  3. Copy all files to your website directory
  4. Include the CSS and JS files in your HTML
  5. The chat button will automatically appear!
Note: You need to configure EmailJS before the form submissions will work. Replace the placeholder credentials below with your actual EmailJS Public Key, Service ID, and Template ID.
